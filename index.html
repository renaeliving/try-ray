Recommended CSV Format

For best results, your schedule upload should include these fields.

Minimum recommended columns
  • Task Name
  • Start Date
  • Finish Date
  • Dependencies or Predecessors
  • Resources
Helpful additional columns
  • Duration
  • Task ID
  • Successors
  • Milestones
  • Baseline Dates
  • Percent Complete
  • Constraints
  • Critical Path Indicators
  • Owner or Team
Example header row
Task ID, Task Name, Start Date, Finish Date, Duration, Predecessors, Resources, Owner, Milestone, Notes

The more complete your schedule export is, the more specific and useful Ray’s feedback will be.
